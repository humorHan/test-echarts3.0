正数与负数
121. 选择题
蜗牛从某点O开始沿东西方向直线爬行，规定向东牌型的路程记作为正数，向西爬行的路程记作为负数，爬行的路程依次为（单位：cm）：+5，-3，+10，-8，-6，+12，-10，若蜗牛在爬行过程中，每爬行1cm，奖励一粒芝麻，则蜗牛一共得到了多少粒芝麻？
正确答案： C ，TA的答案： D ，回答： 错误 。
TA的答案： 未回答 问题的答案
TA作答本题2次，回答错误2次，上次回答是：D
考点： 一元一次方程
难度： 容易
解答： C
试题分析： 根据路线最短的知识可知，两点间的线中直线最短，所以找出直的线即可。
解： 根据分析可知第三条线最直，所以第三条线最短。
笔记： 根据最短路线的知识可知，两点间的线中直线最短，所以找出直线即可。
